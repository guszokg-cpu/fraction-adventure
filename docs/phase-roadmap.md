# Phase Roadmap

## Phase 1: Foundation

สร้างโครง Next.js, layout, theme, mock data และหน้า control center

## Phase 2: Lesson Content

เพิ่มบทเรียนหลัก เช่น รู้จักเศษส่วน อ่านและเขียนเศษส่วน เศษส่วนจากภาพ และเส้นจำนวน

## Phase 3: Exercises and Games

เพิ่ม quiz, เกมจับคู่, เกมระบายเศษส่วน และบันทึกคะแนนแบบ local mock

## Phase 4: Worksheet Builder

สร้างใบงานจากหัวข้อ ระดับชั้น และจำนวนข้อ พร้อม preview

## Phase 5: Classroom Reports

เพิ่ม mock classroom, progress รายคน และ dashboard สำหรับครู

## Phase 6: Supabase

ต่อ auth, database, progress, scores และ RLS เมื่อ UI/data model นิ่งแล้ว

## Phase 7: Responsive and Deploy

ปรับ responsive, accessibility, README, GitHub และ deploy Vercel
