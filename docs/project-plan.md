# แผนโปรเจกต์ Phase 1

เป้าหมาย Phase 1 คือสร้างฐานเว็บแอพให้พร้อมต่อยอด โดยยังไม่ต่อ Supabase และยังไม่ทำฟีเจอร์ทั้งระบบในครั้งเดียว

## ขอบเขต

- Next.js App Router
- TypeScript
- Tailwind CSS
- Layout หลักแบบ desktop 16:9
- Mock data สำหรับผู้ใช้ บทเรียน และภารกิจ
- Theme switcher ที่ผู้ใช้เลือกสีของแอพได้
- หน้า dashboard, lessons, settings และ placeholder สำหรับ module ถัดไป

## ฟีเจอร์ที่ยังไม่ทำ

- Authentication
- Supabase
- ระบบคะแนนจริง
- ตัวสร้างใบงานจริง
- รายงานห้องเรียนจริง
- Responsive เต็มรูปแบบ
