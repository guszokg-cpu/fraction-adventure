# ผจญภัยดินแดนเศษส่วน

เว็บแอพสอนเศษส่วน ป.4-ป.6 ด้วยบทเรียน เกม ภารกิจ ใบงาน และรายงานผล

## Phase 1

- Next.js App Router + TypeScript + Tailwind CSS
- Layout หลักสำหรับ desktop 16:9
- Sidebar, top bar, right panel และหน้า dashboard
- หน้า control center สำหรับตั้งค่าระบบ
- ระบบเลือกธีมสีในเว็บด้วย mock/localStorage
- Mock data สำหรับผู้ใช้ บทเรียน และภารกิจ

## ยังไม่ทำใน Phase 1

- ยังไม่ต่อ Supabase
- ยังไม่ทำระบบ login จริง
- ยังไม่บันทึกคะแนนลงฐานข้อมูล
- ยังไม่ทำ responsive เต็มรูปแบบ

## เริ่มใช้งาน

```bash
npm install
npm run dev
```

เปิด `http://localhost:3000`
